This is a dotnet solution template from projects based on hexagonal architecture and best practices

## Used Technologies

- MediatR
- FluenValidation
- Serilog
- EntityFrameworkCore
- EntityFrameworkCore.SqlServer
- Xunit
- Moq
- Testcontainers
- Autofixture
- FluentAssertions
